Date Sat 18 November 2023 Tags python / code / external

Introduction

This document includes an external Python script to test syntax highlighting in Pelican.

Included Python Code

Below is the content of example_code.py:

# example_code.py

def greet(name):
    """
    Simple greeting function.
    """
    return f"Hello, {name}!"

# Calling the function
print(greet("World"))

Conclusion

The code block above, included from an external file, should be syntax-highlighted if Pygments and Pelican are configured correctly.
